Задача 1421

Построим граф:
1) Имеем исток s и сток t.
2) Каждое предприятие представляет собой пару вершин, соединённых ребром из 1-й во 2-ю.
	Пропускная способность этого ребра равна сумме кредитов этого предприятия.
	1-я вершина соединена с истоком s ребром с бесконечной пропускной способностью.
3) Аналогично с банками.
	Каждый банк представляет собой пару вершин, соединённых ребром из 1-й во 2-ю.
	Пропускная способность этого ребра равна сумме кредито выданных этим банком.
	2-я вершина соединена со стоком t ребром с бесконечной пропускной способностью.
4) 2-е вершины всех предприятий попарно соединены с 1-ми вершинами каждого банками
	ребрами с бесконечной пропускной способностью.
	
Получим граф G(V, E) с пропускной пособностью C.
Необходимо построить максимальный поток и найти его значения на рёбрах между предприятиями и банками.

1) Поток f считаем нулевым для всех рёбер (u, v). Cf остаточная пропускная способность равна C.
2) Ищем путь p из s в t поиском в ширину, такой что для всех рёбер (u, v) из p Сf(u, v) > 0.
	2.1) Если такого нет, то идём на пункт 3.
	2.2) Иначе ищем минимум Cf(u, v) для всех (u, v) из p, обозначим его как Cf(p).
	2.3) Для всех рёбер (u, v) из p
		f(u, v) = f(u, v) + Cf(p),
		f(v, u) = f(v, u) - Cf(p).
	2.4) Для всех рёбер (u, v) Сf(u, v) = С(u, v) - f(u, v).
	Идём на пункт 2.
3) Поток f искомый.

Ответа на задаче нет, если:
Полученый поток меньше суммы всех кредитов (или займов).
Поток на рёбрах между предприятиями и банками более 100.
